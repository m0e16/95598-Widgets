# 95598-Widgets

iPhone 电费查询：Surge 模块 + Scriptable 小组件。

## 南方电网（CSG）

适用于广东、广西、云南、贵州、海南用户：

- **Surge 模块**：捕获南网登录态、定时查电费通知、给小组件提供接口  
- **Scriptable 小组件**：桌面显示余额 / 本月用电 / 近五日  

详见 → [csg/README.md](./csg/README.md)
