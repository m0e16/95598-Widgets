# 快捷指令：自动更新「南方电网.js」

仓库脚本地址：

```text
https://raw.githubusercontent.com/m0e16/95598-Widgets/main/scriptable/csg-widget.js
```

手机路径：`iCloud 云盘 / Scriptable / 南方电网.js`

---

## 方式一：推荐（Scriptable「运行脚本」）

### 1. 安装更新脚本（只需一次）

1. 打开 Scriptable → 新建  
2. 粘贴仓库里的  
   `https://raw.githubusercontent.com/m0e16/95598-Widgets/main/scriptable/update-csg-widget.js`  
3. 保存为：**更新南方电网**

### 2. 做快捷指令

1. 打开「快捷指令」→ 新建  
2. 添加操作：**运行脚本**（Scriptable）  
   - 脚本：更新南方电网  
3. （可选）添加：**显示通知**  
   - 标题：南方电网  
   - 正文：小组件已更新  
4. 快捷指令命名：**更新南方电网小组件**

### 3. 自动跑（可选）

「快捷指令」→ 自动化 → 创建个人自动化，例如：

- **时间**：每天 9:00  
- 或 **打开 App**：打开 Scriptable 时  

动作：运行上面的快捷指令。  
（按提示关闭「运行前询问」）

---

## 方式二：纯快捷指令（不经 Scriptable 运行）

1. 新建快捷指令  
2. **获取 URL 内容**  
   - URL：上面的 raw 地址  
   - 方法：GET  
3. **存储文件**  
   - 询问存储位置：**关闭**  
   - 目标路径：`Scriptable/南方电网.js`  
   - 服务：iCloud 云盘  
   - 覆盖：若文件已存在则覆盖 **打开**  
4. （可选）显示通知  

若路径不对，可先「存储文件」手动选一次 `Scriptable/南方电网.js`，再改成不询问。

---

## 说明

- 更新的是 **脚本源码**；桌面小组件可能要等系统刷新或点一次小组件才显示新版 UI。  
- 需手机能访问 GitHub raw（必要时走代理）。
